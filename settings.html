📖 How to use CK Review Buddy

A quick reference for features that aren't tied to a visible button. Hover any button in the popup for its tooltip.

Text-to-speech on Q-bank pages

Right-click-drag over text — speaks the selection out loud and lights each word on the page as it's read (orange box on the current word, soft grey trail on words already spoken). Works on UWorld, AMBOSS, NBME, and CCS Cases review pages. Only active in review mode so it never triggers during real testing.
Left-click-drag — normal text selection, exactly what your browser already does. No TTS button pops up, and nothing gets auto-highlighted.
Ctrl+Shift+S — speak whatever text is currently selected.
Esc — stop the current read-aloud at any time.

Reading the scraped explanation

In the popup quiz screen, click 🔊 Read NBME Explanation. The extension will locate your Q-bank tab (UWorld / AMBOSS / NBME / CCS Cases) and read the explanation aloud directly on that tab, with the same word-by-word highlighting. If no Q-bank tab is open, it falls back to reading in the popup (no page highlight).

Scraping & quizzing

Scrape current page — pulls the question(s) visible on the active Q-bank tab into CK Buddy. Scrape 1 and Scrape 5 advance through the Q-bank automatically while scraping. Tip: the "Set Start" button marks which question number to begin scraping from if you don't want to restart from the first question.
Start Quiz — after scraping, this turns your missed concepts into spaced-repetition questions. Claude will generate explanations and follow-ups.
Resume Quiz — if you closed the popup mid-quiz, this picks up exactly where you left off. Your progress persists across extension reloads and page refreshes.
Next → — advance to the next quiz question. ⚙ opens per-quiz settings.

Q-bank helpers

AMBOSS Qbank / UWorld Qbank — purple/orange buttons open a specialized review flow for that platform: it reads the current question set and lets you drill into wrong answers by concept.
Question Map (📊) — grid view of every scraped question with status (right / wrong / not attempted). Click a cell to jump.

Troubleshooting

Nothing happens when I right-click-drag
  • TTS is only active in review mode — make sure the question's explanation is visible (not mid-test).
  • On UWorld, make sure you've submitted the answer and are viewing the explanation panel.
  • Try reloading the page after installing a new extension version.
Grey marker rectangles stuck on the page
  • These are the "already-spoken" trail from a TTS pass. They should fade automatically when TTS ends, and get purged when the extension reloads.
  • If any remain, hit Esc or reload the page to clear them.
The extension stopped my quiz from resuming
  • Your quiz state is saved to two keys (ckrb_active_quiz and ckrb_quiz_progress) and should survive reloads.
  • If Resume is missing, open the popup once — the extension migrates older quiz state on first open.
